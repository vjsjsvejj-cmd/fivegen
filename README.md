

#  Five Gen 2.4 - AI 图像与视频生成平台

一个基于 FastAPI + Vue 3 的 AI 图像和视频生成协作平台，内置可视化节点编辑器（Canvas），支持拖拽式连接创作，提供直观的工作流设计体验。

### 视频生成模型（Seedance 2.0）
**Token 计算公式：**
```
Token = ((输入时长 + 输出时长) × 宽 × 高 × 帧率) / 1024
```

**各分辨率尺寸：**
- **480p**
  - 16:9: 864 × 496
  - 4:3: 752 × 560
  - 1:1: 640 × 640
  - 3:4: 560 × 752
  - 9:16: 496 × 864
  - 21:9: 992 × 432

- **720p**
  - 16:9: 1280 × 720
  - 4:3: 1112 × 834
  - 1:1: 960 × 960
  - 3:4: 834 × 1112
  - 9:16: 720 × 1280
  - 21:9: 1470 × 630

- **1080p**
  - 16:9: 1920 × 1080
  - 4:3: 1664 × 1248
  - 1:1: 1440 × 1440
  - 3:4: 1248 × 1664
  - 9:16: 1080 × 1920
  - 21:9: 2206 × 946

**帧率：** 固定 24 帧

## 📺 Canvas 使用指南

### 进入 Canvas
- 点击主界面顶部的「📺 Canvas」按钮
- 直接进入，无需点击5次

### 基础操作
1. **添加节点**
   - 左侧节点选择器点击添加
   - 画布右键快捷添加
   - 支持三种节点：📝 文本提示词、🖼️ 图片上传、🎬 视频上传

2. **节点连线**
   - 从源节点右侧拖出连线
   - 松开鼠标后弹出创建菜单
   - 根据源节点类型显示不同选项
   - 选择后自动创建新节点并建立连线

3. **图生图生成**
   - 连接图片节点 → 选择图生图
   - 输入提示词并配置参数
   - 点击「🚀 生成图片」
   - 结果直接显示在节点上

4. **返回主界面**
   - 点击 Canvas 页面左上角的「← 返回」按钮

### 节点类型说明
- **📝 文本提示词节点** - 输入提示词，可连接到文生图、文生视频
- **🖼️ 图片上传节点** - 上传图片，可连接到图生图、首尾帧、多模态
- **🎬 视频上传节点** - 上传视频，可连接到首尾帧、多模态
- **🖼️ 文生图节点** - 从文本生成图片
- **🎨 图生图节点** - 从图片生成新图片，支持真实 API
- **🎬 文生视频节点** - 从文本生成视频
- **🎞️ 首尾帧节点** - 提取视频首尾帧
- **🌊 多模态节点** - 多模态混合处理

## 🚀 快速开始

### 前置要求
- **Python 3.8+**
- **Node.js 16+**

### 手动启动

#### 1. 启动后端
```bash
cd backend

# 安装依赖
pip install -r requirements.txt

# 配置环境变量（可选，首次运行复制）
# copy .env.example .env

# 启动服务
python main.py
```

#### 2. 启动前端
```bash
cd frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

### 访问地址
- **本地访问**: http://localhost:3000
- **后端 API**: http://localhost:8000

# 火山引擎 API
VOLCENGINE_API_KEY=your-key-here
